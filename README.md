#Ftc Scout Api Tools

This repository holds some useful python scripts which can help out your FTC team during the season.

This fork adds a script which takes as input your team number and lets you select a certain event by it's code and then creates a table which then is exported to a pdf with the matches and teams you have matches with at an event.
There are both REST API and GraphQL(officially used by FtcScout so, it provides the data which is up to date with FtcScout)
I recommend using the GraphQL version because you can be certain that it has the same data that you can see on FtcScout since that is what they use internally.
